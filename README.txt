== Description ==
Inbbe Chat Bot is a wordpress plugin based of an artificial intelligence thats realese the function of a virtual agent. It has been designed to help your customers experiences with a determinated answers. 

If you want to know more of our funcionalities, please look at www.inbbelab.com.


== Installation ==
1. Login to your WordPress admin.
2. Click on the plugins tab.
3. Click the Add New button.
4. Search for \"Inbbe Chat Bot\".
5. Look for our plugin, click \"Install Now\" and then \"Activate\".
6. Once the plugin is installed and active, you simply need to select whether you want to use the default virtual agent of Inbbe Chat Bot or, if you subscribed to our service in inbbelab.com, login with a License to use your determined profile.
